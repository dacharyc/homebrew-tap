# Homebrew Tap

Homebrew formulae for tools by [@dacharyc](https://github.com/dacharyc).

## Install

```bash
brew tap dacharyc/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [skill-validator](https://github.com/dacharyc/skill-validator) | CLI tool that validates and scores Agent Skill packages |

```bash
brew install dacharyc/tap/skill-validator
```
